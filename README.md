# Task Manager REST API

A simple REST API to manage tasks using Node.js and Express.

## 🚀 Setup Instructions

1. Run:
   npm install
2. Start server:
   npm start

Server runs on:
http://localhost:3000

---

## 📌 API Endpoints

POST /tasks  
GET /tasks  
GET /tasks/:id  
PUT /tasks/:id  
PATCH /tasks/:id/done  
DELETE /tasks/:id  

---

## ❗ Error Handling

- 400 → Bad Request  
- 404 → Task not found  
- 405 → Method not allowed
